# Tools

Tools are essential equipment for mining blocks and gathering resources in Starfall. Each pickaxe tier unlocks the ability to mine harder blocks and mines faster than the previous tier.

Equip a pickaxe and press **LMB** on a block to mine it. Use **RMB** to place blocks.

## Tool Progression

Pickaxes follow the same material tier system as weapons. Each tier unlocks mining of blocks at that tier level:

1. **Wood Pickaxe (Tier 0)** -- Mines dirt, sand, wood, and other basic blocks. Craft this first.
2. **Stone Pickaxe (Tier 1)** -- Mines stone and tier 1 ores. Required to progress underground.
3. **Iron Pickaxe (Tier 2)** -- Mines iron, diamond ore, and tier 2 blocks. The mid-game workhorse.
4. **Diamond Pickaxe (Tier 3)** -- Mines titanium ore and tier 3 blocks. Needed for late-game materials.
5. **Titanium Pickaxe (Tier 4)** -- Mines everything, including void stone. The final pickaxe upgrade.

## Specialty Tools

In addition to pickaxes, Starfall has specialized tools for specific tasks:

A fast tool for harvesting trees, leaves, and other soft blocks. Mines at 3x speed but limited to tier 0 blocks. Perfect for gathering large quantities of wood.

### ![](/sprites/item_420.png){width=24} [Boring Drill](/wiki/items/tools/boring-drill)

| Property | Value |
|----------|-------|
| ID | 420 |
| Tier | 2 |
| Stack Size | 5 |
| Crafted At | Tech Bench (10 Iron Bar + 3 Diamond + 20 Stone) |

An automated mining machine. Place it on the ground and it drills 2 tiles wide straight down through the entire world at 12 rows/second, dropping all harvested materials. Explodes when it hits the bottom. Great for quickly accessing deep layers.

### [Shard Compasses](/wiki/items/tools/shard-compasses)

Five endgame compasses (IDs 385-389) crafted at the Void Forge. Each corresponds to a crystal shard type (Ember, Frost, Storm, Void, Life) and helps locate deposits of that shard. Requires Void Dimension materials to craft.

## Mining Tips

- Always carry the best pickaxe you can craft. Mining speed directly affects how quickly you can explore.
- You cannot mine blocks above your pickaxe tier. If a block is not breaking, you need a better pickaxe.
- Upgrade your pickaxe before your weapons. Faster mining means faster access to better materials for everything else.
- Use the Tree Chomper for bulk wood gathering -- it is significantly faster than any pickaxe on trees.
- Drop a Boring Drill near your base for a hands-free shaft to the depths. Collect the material pile when it finishes.
