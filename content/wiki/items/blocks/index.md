# Blocks

Blocks are the building materials of the world in Starfall. The world is made up of over 60 different tile types across multiple layers and biomes. Blocks can be mined with a pickaxe (or bare hands for the softest ones) and placed using right-click.

## Block Properties

- **Hardness tier** determines which pickaxe tier is needed to mine the block and how long it takes.
- Blocks are placed with **right-click** when selected in the hotbar.
- Some blocks are primarily used as crafting ingredients rather than building materials.
